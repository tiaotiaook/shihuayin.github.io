VisualDev UI aims to create an innovative and user-friendly UI library platform that revolutionizes the web development experience by making it more visual and intuitive. OtoDecks

OtoDecks is an audio player for DJs and audio enthusiasts, offering customizable playback with a dual-player system for independent or synced control. It includes essential features like volume control, playback speed adjustment, and track position selection. I also implemented a waveform display for visual monitoring of audio progress, along with custom buttons and design elements to enhance the user experience.

CueMaster

CueMaster is a billiards simulation game I developed using the Matter.js physics engine, aimed at providing users with a realistic billiards experience. Through keyboard and mouse controls, users can simulate striking the ball and experience realistic movement and collisions on the table. The game offers multiple modes, including a standard billiards mode and challenge mode.

Keywords: JavaScript, Matter.js, p5.js, Game Physics

BlogTalk

BlogTalk provides a convenient and intuitive online space for content creators and readers. The platform is developed using the Node.js and Express framework, and it integrates SQLite for managing articles, comments, and user information. The front end uses the EJS templating engine to render dynamic content, ensuring that the interface remains simple and user-friendly throughout the experience.

Keywords: Node.js, Express, SQLite, EJS, RESTful API

TED Persuasive Adjective Explore

This is an analysis project based on TED talk transcripts, aimed at studying the most persuasive adjectives and their sentiment tendencies in TED talks. Using natural language processing techniques, I extracted adjectives from the most popular TED talks of the past decade in fields such as Technology, Science, and Business, and generated word clouds. Additionally, I conducted sentiment analysis on these talks to explore the persuasive adjectives used in different fields and the speech styles they reflect.

Keywords: Web Scraping, Natural Language Processing, Tokenization, Sentiment Analysis, Word Cloud
